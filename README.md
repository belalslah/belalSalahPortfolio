## Usage & Installation

1. Clone the repository:
   ```
   git clone https://github.com/username/belalSalahPortfolio.git
   ```

2. Navigate to the project directory:
   ```
   cd portfolio
   ```

3. Open `index.html` in your web browser or use a local development server:
   ```
   npx serve
   ```
